# ToDoApp
> This is a simple TODO app that coded using PHP language.

# Available Functionalities
- Add a new to do
- Update a to do
- Delete a to do
- Sort by Priority
- View all to do
- Set a to do as completed or uncompleted

Demo: https://smt-todo-app.herokuapp.com/
